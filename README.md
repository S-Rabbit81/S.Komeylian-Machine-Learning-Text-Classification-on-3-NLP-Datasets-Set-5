# In summary: 
# Here, we investigate three methods for converting text into the numerics: 

# ***First Method: Tokenization: 
        # The process of breaking text into individual words or tokens, which both vectorizers rely on.

# ***Second Method: Count Vectorization: 
        # Converts text into a matrix where each column corresponds to a word, and each row corresponds to a document, with the value being the word count.

# ***Third Method: TF-IDF Vectorization: 
        # Normalizes the word counts by the document length to give a measure of how important a word is within a document.
